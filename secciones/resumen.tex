La calidad del aire en espacios cerrados, especialmente en aulas universitarias, es crucial para el bienestar y el rendimiento de los estudiantes. Altos niveles de dióxido de carbono (CO\(_2\)) pueden afectar negativamente la salud y el desempeño cognitivo de los ocupantes. En este contexto, este trabajo presenta el diseño e implementación de un sistema de monitoreo y alerta en tiempo real de los niveles de CO\(_2\) basado en una arquitectura de red de Internet de las Cosas (IoT). El sistema está compuesto por nodos sensores que recopilan mediciones y las transmiten vía comunicación inalámbrica LoRa hacia un gateway personalizado basado en ESP32. Este gateway actúa como puente para enviar los datos mediante Wi-Fi hacia una infraestructura en la nube desplegada en Microsoft Azure. La plataforma web para la visualización y monitoreo de los datos ha sido desarrollada utilizando React para el frontend, acoplada a una API sin servidor alojada en Azure Container Apps, mientras que la información recolectada se almacena en una base de datos NoSQL gestionada con Azure Cosmos DB. El sistema incluye la generación de alertas automatizadas y un subsistema de mitigación activa que acciona ventiladores automáticamente cuando se exceden los umbrales permisibles de CO\(_2\), contribuyendo a la creación de ambientes educativos más seguros y saludables. Las pruebas de integración del sistema validaron el flujo de datos completo y su capacidad para monitorear, alertar y accionar medidas de mitigación en tiempo real.

\noindent CO\(_2\); IoT; LoRa; ESP32; React; Azure; Cosmos DB; monitoreo en tiempo real; mitigación activa; ventilación automatizada.
