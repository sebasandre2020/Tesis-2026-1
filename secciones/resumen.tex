La calidad del aire en espacios cerrados, especialmente en aulas, es crucial para el bienestar y el rendimiento de las personas. Altos niveles de dióxido de carbono (CO\(_2\)) pueden afectar la salud y el desempeño cognitivo de los ocupantes. En este contexto, este trabajo presenta el diseño e implementación de un sistema de monitoreo y alerta en tiempo real de los niveles de CO\(_2\) en aulas mediante el uso de comunicación inalámbrica LoRa en una red de Internet de las Cosas (IoT). El sistema está compuesto por nodos sensores que recopilan y envían datos a un servidor central, donde se almacenan y procesan en la nube utilizando Amazon Web Services (AWS). La plataforma web para la visualización y monitoreo de datos ha sido desarrollada utilizando .NET y la información recolectada es almacenada en una base de datos gestionada con SQL Server. El sistema incluye la generación de alertas cuando se exceden los niveles permitidos de CO\(_2\), contribuyendo a la creación de ambientes educativos más seguros y saludables. Las pruebas del sistema en aulas seleccionadas validaron su capacidad para monitorear y alertar en tiempo real.
