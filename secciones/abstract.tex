\customchapter{ABSTRACT}

Air quality in enclosed spaces, especially in university classrooms, is crucial for the well-being and performance of students. High levels of carbon dioxide (CO\(_2\)) can negatively affect the health and cognitive performance of occupants. In this context, this work presents the design and implementation of a real-time monitoring and alert system for CO\(_2\) levels based on an Internet of Things (IoT) network architecture. The system is composed of sensor nodes that collect measurements and transmit them via LoRa wireless communication to a custom gateway based on the ESP32 microcontroller. This gateway acts as a bridge to send the data via Wi-Fi to a cloud infrastructure deployed on Microsoft Azure. The web platform for data visualization and monitoring was developed using React for the frontend, coupled with a serverless API hosted on Azure Container Apps, while the collected information is stored in a NoSQL database managed with Azure Cosmos DB. The system includes the generation of automated alerts and an active mitigation subsystem that automatically triggers fans when permissible CO\(_2\) thresholds are exceeded, contributing to the creation of safer and healthier educational environments. Integration tests of the system validated the complete data flow and its ability to monitor, alert, and actuate mitigation measures in real-time.

\noindent \textbf{Keywords:}\\
\noindent CO\(_2\); IoT; LoRa; ESP32; React; Azure; Cosmos DB; real-time monitoring; active mitigation; automated ventilation.
